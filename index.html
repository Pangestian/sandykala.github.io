Cara masuk Server Sandykala


1. Download 2 Modpack di menu 'Modpack'.
2. Install Minecraft 1.20.1 dengan Modloader Forge terbaru!
3. Taruh file modpack utama dan modpack fps boost di folder mods,
4. Ekstrak agar semua file .jar di dalam file .zip ada di folder mods.
5. Jika anda player Pojav, gunakan renderer selain Default, direkomendasikan LTW.
6. Launch Game nya!
7. Masuk kan IP server Sandykala, and enjoy the server!
